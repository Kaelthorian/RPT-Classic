# Instructions for `Icons`

- DMI and related resources are binary assets. Do not scan or modify them during normal
  code tasks.
- Preserve exact file names, casing, spaces, icon states, dimensions, and paths.
- Edit only when the user explicitly requests an asset change and use an appropriate
  asset-aware tool. Never mass-convert or optimize the directory.
- After an explicit resource change, compile and verify loading/rendering in BYOND.
